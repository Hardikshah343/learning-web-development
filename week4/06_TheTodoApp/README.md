# TheTodoApp

A simple TODO application with following features:
* A user can signup
* A user cas sign in 
* A user can create a todo
* A user can see their exisiting todos
* A user can mark a todo as done

## Backend
1. Initialise a node project
2. Put a package.json
```terminal

version: (1.0.0)
```
